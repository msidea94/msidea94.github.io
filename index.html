Popup

Popup Btn

배달주문

주소입력

검색

현재위치로 주소검색하기

  • - 도로명 + 건물번호(예:국회대로 74길 4)
  • - 동/읍/면/리 + 번지(예:여의도동 13-13)
  • - 구/동명 + 건물명(아파트명)(예: 대치동 서경빌딩, 강남구 서경빌딩)

검색결과(주소를 선택하세요)

close

HTML

<div class="popup-btn" data-name="data-order">
<a href="#" class="btn type01 btn-l-line-red">배달주문</a>
</div>
<div class="popup-content" data-name="data-order">
<p >컨텐츠내용</p>
</div>

- popup 버튼 class="popup-btn" 고정입니다.
- 팝업버튼 자식 a 요소 고정입니다.
- 팝업컨텐츠 class="popup-content" 고정입니다.
- 팝업버튼과 팝업컨텐츠 속성 data-name=""의 value는 두곳 똑같이 정합니다.
